### Setup Formspree (5 minutes):

1. Go to https://formspree.io/ and sign up (FREE)
2. Create a new form
3. Copy your form ID
4. Open `index.html` and find this line (around line 698):
   ```html
   <form class="contact-form" id="contactForm" action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
5. Replace `YOUR_FORM_ID` with your actual Formspree ID
6. Commit and push:
   ```bash
   git add index.html
   git commit -m "Add Formspree form ID"
   git push
   ```

**That's it! Your contact form will work!**
