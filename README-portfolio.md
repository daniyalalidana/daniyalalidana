## ğŸ¨ Customization

### Update Personal Links

Find and replace these placeholders in `portfolio.html`:

- `https://github.com/daniyalalidana` - Your GitHub profile
- `https://linkedin.com/in/daniyalalidana` - Your LinkedIn profile

### Modify Colors

Edit the CSS variables in `portfolio-styles.css`:

```css
:root {
    --primary-color: #6366f1;      /* Main brand color */
    --secondary-color: #ec4899;    /* Accent color */
    --accent-color: #14b8a6;       /* Highlight color */
}
```
